This repository contains the code for the [PurseStrings.com](https://pursestrings.com) domain-for-sale landing page.

## Purpose

This landing page notifies visitors that PurseStrings.com is available for purchase and provides an easy way for interested buyers to submit inquiries.

## Features

- Modern, mobile-friendly single-page design
- Clearly advertises that PurseStrings.com is for sale
- Integrated inquiry form (supports Formspree or your chosen provider) to collect offers and questions
- Automatically detects and displays the current domain name on the page and in form submissions
- Fast, secure, and straightforward deployment (static HTML, CSS, and JavaScript)
- Easily customizable for content and appearance

## Usage

1. **Clone or download this repository.**
2. **Customize the content** in `index.html` as needed (domain name, form endpoint, features, etc.).
3. **Deploy to your preferred static hosting provider** (Netlify, Vercel, GitHub Pages, cPanel, etc.).
4. **Update DNS settings** to point PurseStrings.com to your hosting provider.

## Customization

- Update the domain name in headers, meta tags, and throughout the landing page as desired
- Set the Formspree endpoint or other contact integration for form submissions
- Adjust styles, features, and text to fit your preferences or brand

## DNS Setup

For best results, point PurseStrings.com to your hosting provider using a CNAME, ALIAS/ANAME, or A record as suitable.  
Avoid HTTP redirects, which can interfere with domain detection and inquiry tracking.
